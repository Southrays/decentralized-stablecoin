This is a DecentralizedStablecoin system
The DecentralizedStablecoin has the following properties:
 1. Collateral: Exogenous:
    Backed by wEth & wBtc.

 2. Relative Stability:
    Pegged - 1:1 with the usd.

 3. Minting:
    Algorithmic.

This Stablecoin must always be over collateralized. The $ value of all the collaterals must always be more
than the $ value of all the minted DecentralizedStablecoin DSC

Any user who goes below the liquidation threshold of the DSC System can be liquidated
